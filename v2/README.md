# Mohio v2 Prototype

`v2` is a standalone `Vite + React + TypeScript` prototype for Mohio Phase 1: the core workspace for a Markdown-based, AI-assisted team wiki.

## What it includes

- Personal scratchpad with quick jot capture
- Shared wiki pages with lightweight Markdown editing modes
- AI proposal review with readable diffs
- Draft staging before publish
- Explicit publish review
- Page-level history and contextual related-page navigation

Everything is local and seeded in memory. There is no backend, filesystem sync, or real AI integration in this prototype.

## Run

```bash
cd v2
npm install
npm run dev
```

Then open the local Vite URL.

## Test

```bash
cd v2
npm test
```
