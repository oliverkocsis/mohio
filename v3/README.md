3# Mohio v3 Prototype

`v3` is a standalone `Vite + React + TypeScript` prototype for the Mohio flagship concept: a calm desktop workspace that blends a Notion-like page canvas with a Codex-style AI sidecar.

## What it includes

- Page tree navigation with connected Mohio documents
- Block-based document canvas with inline editing
- AI thread, quick actions, and suggestion cards
- Draft-first trust boundary for applied AI changes
- Lightweight recent history and activity log

Everything is local and seeded in memory. There is no backend, no real AI, and no filesystem sync in this prototype.

## Run

```bash
cd v3
npm install
npm run dev
```

## Test

```bash
cd v3
npm test
```

## Build

```bash
cd v3
npm run build
```
