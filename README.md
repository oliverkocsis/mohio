# Mohio

know, understand, do

## Know

Mohio is the unit of knowledge.
Like atom is the unit of matter.
Mohio can be a fact, a name, a date, a place, a piece of information, a vocabulary, or anything else ...

## Understand</h2>

Everything is connected.
Every information is composed of mohios.
This relationship is infinite, has no begining, has no end.
This relationship defines big ideas, generalizations, principles, ideas that transfer across situations, and much more ...

## Do
Understanding provides the ability to use the knowledge effectively and readily in execution or performance.
