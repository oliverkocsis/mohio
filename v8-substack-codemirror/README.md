# Mohio v8 Substack CodeMirror Prototype

`v8-substack-codemirror` is a standalone browser-only Markdown editor prototype built on the `v6-codemirror` foundation and restyled as a Substack-like article editor.

## Run

```bash
cd v8-substack-codemirror
npm install
npm run dev
```

## Test

```bash
cd v8-substack-codemirror
npm test
```

## Build

```bash
cd v8-substack-codemirror
npm run build
```
