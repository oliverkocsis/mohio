# Mohio v4 Prototype

`v4` is a simplified full-screen desktop prototype for Mohio's core loop:

- capture a jot,
- review a proposal,
- turn it into shared wiki updates.

The app keeps the calm Mohio visual language but removes the heavier `v2` workflow surfaces. It is local-only and uses seeded in-memory data.

## Run

```bash
cd v4
npm install
npm run dev
```

## Test

```bash
cd v4
npm test
```
